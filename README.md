
Приложение для Windows (WPF, .NET 8) для создания шаблонов конвертов, ведения базы получателей и печати с предпросмотром. 
Поддерживает выбор принтера/лотка/типа бумаги, настройку масштаба и смещений, диапазоны печати, полноэкранный предпросмотр, импорт/экспорт Excel и автоматические миграции БД.

Возможности

- Дизайнер шаблонов: создание и редактирование элементов шаблона (текст/изображения), слои, задний фон.

- Предпросмотр печати: масштаб, ориентация, формат бумаги, проверка печатной области с учетом полей принтера.

Печать:

- выбор принтера, формата, лотка (InputBin) и типа бумаги (MediaType);

- диапазоны страниц: «Все», «Текущая» или выражение (например, 1-3,5,8-10);

- копии, вписывание в печатную область или явный масштаб/смещения.

База получателей (SQLite, EF Core): добавление/редактирование, фильтры, сортировка.

- Импорт/экспорт Excel (ClosedXML).

- Системные диалоги/окна: ошибки, ввод значений, прогресс печати, полноэкранный просмотр.

- Автомиграции БД и обслуживание (WAL checkpoint, backup/restore).

- Локаль приложения: ru-RU.

Требования

- Windows 10 2004 (10.0.19041) или новее.

- .NET SDK 8.0+

- Visual Studio 2022 с рабочей нагрузкой «.NET Desktop development».

- Принтер (локальный или сетевой) с установленными драйверами.
